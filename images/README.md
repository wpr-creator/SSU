# SSU Image Assets

Images for **US History** are organized by semester and lesson folder.

## Folder Structure

```
images/
  S1/
    L01/ … L08/
  S2/
    L01/ … L08/
```

Day 9 (final exam) does not have a dedicated image folder.

## File Naming

Use this format:

**`{Semester}{Lesson}_DescriptiveName.jpg`**

| Part | Example | Meaning |
|------|---------|---------|
| Semester | `S1` | Semester 1 |
| Lesson | `L01` | Lesson / day 1 |
| Descriptive name | `Constitution` | Short description of the image |

**Full example:** `images/S1/L01/S1L01_Constitution.jpg`

Use clear, descriptive names. Keep filenames consistent so they are easy to find when building lessons.
